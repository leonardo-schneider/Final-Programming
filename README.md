# Young People Survey EDA

An exploratory data analysis of the "Young People Survey" dataset. This project investigates correlations between music preferences, movie genres, hobbies, and demographic variables to identify clusters and behavioral patterns.

## Project Structure

- `eda_young_people_survey.ipynb`: The main Jupyter notebook containing the analysis and visualizations.
- `pyproject.toml` & `uv.lock`: Dependency definitions for reproducible environments.

## Prerequisites

- **Python:** 3.12+
- **Environment Manager:** uv, or anything which can install packages with a requirements.txt file.
